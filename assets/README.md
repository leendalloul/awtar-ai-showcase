# Product Visuals

This directory contains approved, publication-ready Awtar AI product visuals.

Published visuals:

- `dashboard.png` — operations dashboard and business health overview
- `team-handoff.png` — human-in-the-loop workflow for missing information

Do not add raw or unreviewed screenshots. Before publishing an image:

1. Use fictional or explicitly approved demonstration data.
2. Complete an internal privacy and publication review.
3. Review the image at full resolution.
4. Remove customer data, credentials, identifiers, internal URLs, browser chrome, logs, and implementation details.
5. Confirm that cropping or blurring cannot expose recoverable sensitive content.

No production screenshot should be assumed safe without a deliberate review.
